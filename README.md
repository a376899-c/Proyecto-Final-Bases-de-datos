# UniRide 🚗

Sistema desarrollado para facilitar la movilidad entre estudiantes universitarios mediante viajes compartidos.

## Funciones del sistema

- Registro de pasajeros
- Registro de conductores
- Inicio de sesión
- Publicación de viajes
- Pagos
- Calificaciones

## Tecnologías utilizadas

- Java
- NetBeans
- Oracle Database XE
